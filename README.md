# Abiral Senchuri — Portfolio

A personal portfolio website built using plain HTML, CSS, and JavaScript.

## Features
- Responsive design
- Dark mode UI
- Smooth animations
- Project showcase
- Contact section

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Files
- `index.html` → main page
- `styles.css` → styling
- `script.js` → functionality
- `data.js` → portfolio data

## Run Locally
Just open `index.html` in your browser.
